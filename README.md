# csgo-external
Old and discontinued external hack for Counter-Strike: Global Offensive.
## Features
* Aimbot
    - Key
    - FOV
    - Smooth
    - Bone
    - Height modificator
* Triggerbot
    - Key
    - Delay
* Recoil Control System (RCS)
    - Scale
    - Standalone mode
* Glow ESP
    - Show friendly
    - Health-based glow
* Bhop
* No flash
* Config system
